HyperText Markup Language (HTML) Tutorial

This tutorial covers how to make a page, and many elements you can put onto your very own webpage.


Start here

Introduction to HTML

HTML is the language used to create webpages. It has lots of different tags to help developers create what they want

Creating an index.html file

On Windows, create a file in Notepad, and save it as index.html
On MacOS, open a new file in TextEdit. Save this file as index.html

Start coding

Let's start with how to create tags. Tags are defined inside <>. For example, <html>. To close an element, simply include a / at the beginning of the tag, like this </html>.

So, let's open our index.html files.

Getting Started with HTML Coding

Our index.html files should be empty, so lets fix this by adding some html.

<html>
</html>

These two tags define the html files. All html code will go between these two tags. The next two tags are <head> and <body>. These go inside the <html> tags.

Our code should now look like this

<html>
<head></head>
<body></body>
</html>

Within the <head> and <body> tags, we can add more to increase the functionality of our webpage.
Add the <title> tag inside the <head> elements to add a title to your webpage, which will show in the tab bar. It goes within the head tag as shown below.

<head>
<title>Your webpage title goes here</title>
</head>
